> 2048-Game, my first repository

2048-Game 是我**第一个也是开发最久**的GitHub仓库

现在它已经更新到了 **v2.5.9(160)** 版本，版本迭代很快

(*它曾经还是我的个人主页*)

## 这个项目的由来

在我刚注册完 GitHub 后就关注到了原版 [gabrielecirulli](https://github.com/gabrielecirulli/2048) 的开源2048

我当时非常感兴趣，于是我就下载了下来，开始研究

但由于种种原因，直到4月中旬，我才正式开始编辑

半个月之后，我终于完成了这个项目，并推送到了GitHub上

## 项目细节

最开始我其实是想用原生 JavaScript 实现的

带感觉有点难度，所以使用 JQuery 动画实现了

---

确定使用 jQuery 之后，我就开始疯狂编辑

18号的时候，我就基本做完了初始代码，然后开始**疯狂调试**

调得基本差不多的时候，我就尝试把代码分开来，弄完后就开始翻译**30种语言**

---

这些都搞完后，我又开着别人的**彩蛋设置页面**，感觉非常的*cool~*，

所以我也加进来了，并对它进行了一些优化(*实际上改了很多*)

---

这些都差不多搞完后,我就把项目推送到了 GitHub 上面，开始实际页面调试

上传完后，我在应用商店里面看到一个AI工具—— TRAE 。

据说他可以直接连接你的 GitHub 仓库，于是我就下载来使用了一下

效果也是非常的好，帮我修复了很多 BUG，在这里也推荐你们用一下

---

搞了这些以后，我就陆陆续续的更新了一些东西(*比如WASD操控*)

随后我就开始静默更新了(*相当于已经存档了*)

---

>更多细节，你可以看看我的[更新日志](https://github.com/bill74186/2048-Game/blob/main/description.md)

## 使用方法

### 1. 直接访问 2048-Game

点这里→ [2048-Game](https://bill74186.github.io/2048-Game)

### 2. 下载游玩

```
git clone git@github.com/bill74186/2048-Game.git
cd 2048-Game
python3 -m http.server 8080
# Press "Ctrl + C" to stop
```
